# 🔧 문제 해결 가이드

## 분석 결과가 표시되지 않는 문제

### 문제 증상
- 정책 문서를 붙여넣으면 "분석 중..." 텍스트만 표시됨
- 한참 기다려도 분석 결과가 나오지 않음

### 해결 방법

#### 1. 기본 분석 결과 즉시 표시
- ✅ 기본 필터링 결과는 즉시 표시됩니다
- ✅ LLM 분석은 백그라운드에서 진행됩니다

#### 2. 타임아웃 설정
- LLM 분석은 25초 타임아웃이 설정되어 있습니다
- 타임아웃 시 기본 분석 결과만 표시됩니다

#### 3. API 키 설정 확인
- Vercel 대시보드에서 환경 변수 확인
- `OPENAI_API_KEY`가 설정되어 있는지 확인
- API 키가 없어도 기본 분석은 정상 작동합니다

## 일반적인 문제 해결

### 1. 빌드 오류
```bash
# 로컬에서 빌드 테스트
npm run build
```

### 2. API 함수 오류
- Vercel 대시보드 → Functions 탭에서 로그 확인
- 환경 변수 설정 확인

### 3. 네트워크 오류
- 브라우저 개발자 도구 → Network 탭 확인
- `/api/analyze` 요청 상태 확인

## 디버깅 팁

### 브라우저 콘솔 확인
1. F12 또는 Cmd+Option+I (Mac)로 개발자 도구 열기
2. Console 탭에서 오류 메시지 확인
3. Network 탭에서 API 요청 상태 확인

### Vercel 로그 확인
1. Vercel 대시보드 → 프로젝트 선택
2. Functions 탭 → `/api/analyze` 클릭
3. 로그에서 오류 메시지 확인
